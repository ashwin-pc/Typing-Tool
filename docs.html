How to use

There are 3 functions available to use in the typing tool class

  1. .type()
  2. .erase()
  3. .eraseType()

Typing tool class TypingTool(options)

Options

  • container: The Dom element to perform the typing action on (default: document.body)
  • speed: The default rate at which the typing should occur (default: 500)
  • delay: The default delay before performing any action (default: 0) 
 var typing = new TypingTool();
or 
 var typing = new TypingTool({
    container: "containerID",
    speed: 100,
    delay:0
 });

Type function .type(text, options)

This function is used to simulate the typing of a new line onto a given container

text: is the string you want to be simulate

 typing.type("Hello World")

Options are the same as that of the class initialization. However these options will affect only the current typing instance.

Erase function .type(partialLength, options)

This function is used to simulate the erasing of a line of text in a given container. Partial erasing of a text is also possible.

partialLength: length of the text to remove from the container from the end

 typing.erase()

Options are the same as that of the class initialization. However these options will affect only the current typing instance.

Erase and Type function .eraseType(text, options)

This function is used to simulate the erasing followed by typing of a line of text in a given container. If no text is present, it will just type.

text: is the string you want to be simulate

 typing.eraseType("Hello World")

Options are the same as that of the class initialization. However these options will affect only the current typing instance.
